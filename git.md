# Система контроля версий git

## Что такое git?
Git - это система контроля версий.
## Подготовка репозитория
Команда git init
## Создание "сохранений"
Команды git add "__"
git commit -a -m "Description"
## Перемещение между сохранениями
Команда git checkout __
## Журнал изменений
Команда git log
